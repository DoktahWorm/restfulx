ruboss4ruby

Get Version

1.0.3

What

Ruboss Framework Integration Support for Rails 2.+ and Merb 0.9.3+

Installing

sudo gem install ruboss4ruby

Forum

http://groups.google.com/group/ruboss-framework

Source

You can fetch the source from:

git clone git://github.com/dima/ruboss4ruby.git

Build and test instructions

cd ruboss4ruby
rake test
rake install_gem
