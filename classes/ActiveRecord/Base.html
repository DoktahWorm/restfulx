includes_as_hash(includes = nil)

options[:include] can be a Hash, Array, Symbol or nil. We always want it as a Hash. This translates includes to a Hash like this: If it‘s a nil, return an empty Hash ({}) If it‘s a Hash, then it is just returned If it‘s an array, then it returns a Hash with each array element as a key, and values of empty Hashes. If it‘s a symbol, then it returns a Hash with a single key/value pair, with the symbol as the key and an empty Hash as the value.
